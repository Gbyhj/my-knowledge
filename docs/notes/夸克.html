夸克

五槽

  • **包含槽**：{ 上夸克, 下夸克, 粲夸克, 奇夸克, 顶夸克, 底夸克 }
  • **量化槽**：6种味 × 3种色 = 18种；自旋 1/2；电荷 +2/3 或 -1/3
  • **状态槽**：色禁闭（单独夸克不存在，只以强子形式存在）
  • **功能槽**：通过强相互作用（胶子）结合成强子
  • **关系槽**：与轻子同属费米子；与胶子通过色荷耦合

    • 笔记

  • 夸克不能单独存在——**色禁闭**
  • 质子的组成：uud；中子的组成：udd
  • 胶子是传递强相互作用的规范玻色子
