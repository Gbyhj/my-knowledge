五槽模型





对象 = {


① 包含槽     {紧邻下层对象}


② 量化槽     数值 + 时间 + 同义词


③ 状态槽     当前状态 + 可能状态


④ 功能槽     ~400 原子动作


diff 槽      可选：参考对象 + 差异


⑥ 关系槽     同层关联，矛盾可共存


}

包含关系



L0 包含 → L1（强子由夸克组成）


L1 包含 → L2（原子核由强子组成）


L2 包含 → L3（原子由原子核和电子组成）


L3 包含 → L4（分子由原子组成）


L4 包含 → L5（物质由分子组成）


...依此类推



每一层是紧邻下层的高阶组合。
