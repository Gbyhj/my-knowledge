科学

五槽

  • **包含槽**：{ 物理学, 化学, 生物学, 天文学, 地球科学, 计算机科学, 社会科学 }
  • **量化槽**：基于观察、实验、可重复验证的系统性知识体系
  • **状态槽**：不断修正，不断进步
  • **功能槽**：理解和预测自然现象，转化为技术
  • **关系槽**：← 数学, 哲学；→ AI与未来

    • 科学方法

    1. 观察 → 2. 提出假说 → 3. 做出预测 → 4. 实验检验 → 5. 修正/推翻假说

    关键特征

  • **可证伪性**（波普尔）：一个理论必须是可能被证明错误的
  • **范式转换**（库恩）：科学革命是世界观的整体变迁
  • 科学 ≠ 真理，而是**最可靠的当前理解**
