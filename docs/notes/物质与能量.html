物质与能量

五槽

  • **包含槽**：{ 单质, 化合物, 有机物, 热量, 声子 }
  • **量化槽**：质量守恒；能量守恒
  • **状态槽**：固/液/气/等离子体；温度/压强决定相态
  • **功能槽**：构成可感知的物理世界
  • **关系槽**：← 分子；→ 2-生命

    • 关键概念

  • **单质**：由同种原子组成（O₂, Fe, C）
  • **化合物**：由不同原子组成（H₂O, NaCl）
  • **有机物**：含碳化合物（生命的基础）
  • **热量**：分子热运动的能量传递
  • **声子**：晶格振动的量子化准粒子
  • **E = mc²**：质量和能量等价
