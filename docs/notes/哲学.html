哲学

五槽

  • **包含槽**：{ 形而上学, 认识论, 伦理学, 逻辑学, 美学 }
  • **量化槽**：系统性批判性思维
  • **状态槽**：永恒的追问，无终结答案
  • **功能槽**：追问根本问题：存在、知识、价值、理性、意义
  • **关系槽**：← 人类, 语言；→ 科学, 伦理

    • 分支

    分支核心问题 形而上学存在什么？实在的本质？ 认识论我们如何知道？真理是什么？ 伦理学何为善恶？应该如何生活？ 逻辑学有效推理的规则是什么？ 美学美是什么？艺术是什么？

    主要流派

  • 理性主义（笛卡尔、斯宾诺莎、莱布尼茨）
  • 经验主义（洛克、贝克莱、休谟）
  • 康德：先天综合判断
  • 分析哲学（弗雷格、罗素、维特根斯坦）
  • 存在主义（海德格尔、萨特、加缪）
